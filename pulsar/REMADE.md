# Pulsar

## Introduction

Pulsar is a multi-tenant, high-performance solution for server-to-server messaging. It is a distributed messaging system that is designed for high-performance, availability, and scalability. It is a messaging system that is built on top of Apache BookKeeper and can be used to build messaging systems that are highly available, scalable, and fault-tolerant.

- [Official Website](https://pulsar.apache.org/)
- [Documentation](https://pulsar.apache.org/docs/3.3.x/)
- [GitHub](https://github.com/apache/pulsar)
